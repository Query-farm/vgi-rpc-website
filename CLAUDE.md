# CLAUDE.md — vgi-rpc-website

## Project overview

Documentation/marketing website for the vgi-rpc framework. Built with **Astro 5 + Tailwind CSS 4**, deployed to **Cloudflare Pages**.

Live site: https://vgi-rpc.query.farm

## Commands

```bash
npm run dev        # Generate diagrams + start dev server
npm run build      # Generate diagrams + production build → ./dist/
npm run preview    # Preview production build locally
npm run diagrams   # Generate D2 diagrams to SVG (requires `d2` CLI)
```

## Project structure

```
src/
  pages/           # Astro file-based routing (index, wire-protocol, benchmarks)
  components/      # Reusable .astro components (Hero, Nav, Footer, etc.)
  layouts/         # BaseLayout.astro — HTML wrapper, Open Graph, JSON-LD
  styles/          # global.css — Tailwind imports + custom theme (farm-* colors)
  data/            # capabilities.json (auto-generated by CI, do not hand-edit)
  diagrams/        # D2 diagram source files
public/            # Static assets (logos, generated diagram SVGs, llms.txt)
scripts/           # test-capabilities.py — CI script that probes language implementations
```

## Key conventions

- **Astro components**: frontmatter (TypeScript) between `---` fences, template below. Props use `interface Props`.
- **Styling**: Inline Tailwind classes. Dark mode via `dark:` prefix. Responsive mobile-first (`sm:`, `md:`, `lg:`).
- **Theme colors**: Custom CSS variables (`farm-green-*`, `farm-gold`, `farm-cream`, `farm-dark`, etc.) defined in `src/styles/global.css`.
- **Fonts**: Inter (body), JetBrains Mono (code) from Google Fonts.
- **External links**: Always use `target="_blank" rel="noopener noreferrer"`.
- **Brand**: Use lowercase `vgi-rpc` in monospace when referring to the framework.

## Data flow

- `scripts/test-capabilities.py` probes language repos (Python, TypeScript, Go, C++) for transport/pattern/feature support.
- Output goes to `src/data/capabilities.json` — consumed by `CapabilityMatrix.astro`.
- This runs weekly via GitHub Actions and auto-commits changes.

## CI/CD

- **Deploy** (push to main + weekly): Node 22, installs D2, runs `npm run build`, deploys via Wrangler.
- **Capability update** (weekly + manual): Python 3.12, Go 1.22, Bun — runs `test-capabilities.py`, auto-commits if changed.

## Dependencies

Minimal: `astro`, `@astrojs/sitemap`, `tailwindcss`, `@tailwindcss/vite`. No runtime JS frameworks.
